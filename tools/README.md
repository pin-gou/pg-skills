# pg-skills 开发工具

| 工具 | 路径 | 用途 |
|---|---|---|
| 📋 project-editor | `project-editor/` | `.pg/project.yaml` 图形化编辑器 (新版, 4 视图 + 100% 段覆盖 + Diff 确认) |
| 📋 yaml-editor | `yaml-editor/` | `.pg/project.yaml` 图形化表单编辑器 (旧版, 保留为回退) |

## 启动 project-editor (推荐)

```bash
cd project-editor && pnpm install && pnpm dev
# 浏览器 http://localhost:3028/
```

详见 `project-editor/README.md`。

## 启动 yaml-editor (旧版)

```bash
cd yaml-editor && pnpm install && pnpm dev
# 浏览器 http://localhost:3028/
```
